TaskMate frontend is the user interface of the system.  
It is built using **Next.js**, **React**, and **Tailwind CSS**.  
The frontend connects with the Laravel backend using APIs.

---

## Features

### Authentication
- User Login
- User Registration
- Token-based login

### Dashboard
- Login with Google
- Login with Facebook
- View all tasks
- Add new task
- Edit task
- Delete task

### Profile Management
Users can:
- View profile
- Update username
- Update email
- Logout

---

## Requirements

Make sure the following are installed:

- Node.js 18
- NPM
- Git

---

## Project Installation

### Run these commands to install the frontend

- npm install
- Create `.env.local` file
- Add backend API URL in `.env.local`
- npm run dev
